# CricQuiz 

## Description- 
A fun quiz app that will test you knowledge about different cricketing trophies. 

## Technologies Used - 
- HTML
- CSS
- Components are taken from [Frid](https://frid-ui.netlify.app)

## Pages included-
-Landing Page
-Categories 
-Rules 
-Questions
-Answers and Result
